Introduction to Html

HTML which is short for Hypertext Markup Language is a document that is used to diplay in a web browser. It was first proposed by physicist Tim Berners-Lee in 1980 and introduced in late 1989. It is links that connect web pages to one another, either within a single website or between websites. World first website

It allowed publishing and exchanging of scientific and technical documents. Since then the web have grown a lot and html is widely being used in all the webpages. The main concept of HTML was to have the connected documents which it enabled by having hypertext, which means now many files can be connected and accessed via one file. According to MDN description HTML uses "markup" to annotate text, images and other content for display in a web browser.

An HTML document is a plaintext document structured with elements. Elements are surrounded by matching opening and closing tags. Each tag begins and ends with angle brackets (<>). Browsers do not show HTML tags, but use them to describe the contents of the page.

< h1 > Hello World< /h1 >

< p > I am going for walk in few minutes.< /p >

You can extend the HTML tags with attributes that provide additional information on how the browser interprets the element. Attributes such as < img > and < input > used to provide image and input. Most of the tags have closing tag and some don't.

< img src=" Your picture here" >

< a href="link you want to add" > < /a >

Inorder to diplay the list, we must know there are two types of lists in HTML. Order list which are the list of numbers and Unorder list which list without number.

< ul > < li > Fruits < /li > < /ul >

< ol > < li > Vegetables < /li > < /ol >
